sqlcmd 사용법 (sqlcmd-guide)

sqlcmd

대용량 sql 파일실행은 sqlcmd로만 실행가능하다.
다른 sql 실행방법은 메모리에 해당 sql 파일을 로딩한 상태로 실행하여 크기제한이 있지만 sqlcmd는 해당 sql문 하나하나 별도로 실행하기 때문이다.

사용 예시

sqlcmd \
-S 127.0.0.1,1433 \
-i input_file[ , input_file2...] \
-o output_file \
-U login_id \
-P password \
-d db_name

-S : 접속할 IP,접속포트
-i : 실행할 sql 파일 지정
-i C:<filename>
-i \<Share$><filename>
-i “C:\Some Folder<file name>”
-o : 실행결과 저장할 파일 지정
-U : 계정
-P : 패스워드
-d : Database 이름

windows 환경

  • 계정과 패스워드는 설정해주지 않으면 윈도우 기본계정으로 접속

linux 환경에서 대용량 파일 줄단위 분할

  • split -l 10000 “aaa.sql” “aaa.sql.”
